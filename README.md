# Tirajosaure

Tirajosaure est une application de tirage au sort et de création d'équipe développé sur iOS 16 minimum en SwiftUI et utilisant Back4App comme base de donnée.

## Connexion/Inscription

Tirajosaure utilise Back4App comme base de donnée et permet la création d'utilisateurs pour les enregistrer et les connecter.

<img src="https://github.com/Bananathoms/OCP12-Tirajosaure/raw/main/ReadMe/Login.png" alt="Login" width="300"/>
<img src="https://github.com/Bananathoms/OCP12-Tirajosaure/raw/main/ReadMe/Signup.png" alt="Signup" width="300"/>

Back4app intègre également un système de réinitialisation de mot de passe permettant à l'utilisateur de recevoir un e-mail afin de pouvoir modifier son mot de passe.

<img src="https://github.com/Bananathoms/OCP12-Tirajosaure/raw/main/ReadMe/password.png" alt="Password" width="300"/>

## Question

Possibilité de créer de nouvelles questions possédant plusieurs options parmi lesquels un tirage au sort pourra être effectué.

<img src="https://github.com/Bananathoms/OCP12-Tirajosaure/raw/main/ReadMe/addQuestion.png" alt="Add Question" width="300"/>
<img src="https://github.com/Bananathoms/OCP12-Tirajosaure/raw/main/ReadMe/question.png" alt="Question" width="300"/>

Les questions sont modifiables à tout moment avec l'option d'édition. Un historique et des statistiques sont disponibles pour les tirages effectués dans la question.

<img src="https://github.com/Bananathoms/OCP12-Tirajosaure/raw/main/ReadMe/questionHistoric.png" alt="Question Historic" width="300"/>

## Évènement

Possibilité de créer des événements possédant des équipes et des membres afin de répartir les membres dans les différentes équipes de l'événement.

<img src="https://github.com/Bananathoms/OCP12-Tirajosaure/raw/main/ReadMe/event1.png" alt="Event 1" width="300"/>
<img src="https://github.com/Bananathoms/OCP12-Tirajosaure/raw/main/ReadMe/Event2.png" alt="Event 2" width="300"/>
<img src="https://github.com/Bananathoms/OCP12-Tirajosaure/raw/main/ReadMe/event3.png" alt="Event 3" width="300"/>
